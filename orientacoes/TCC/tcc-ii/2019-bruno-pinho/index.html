Resumo

O número de Grundy é maior quantidades de cores que o algoritmo guloso de coloração consegue atribuir a um grafo. Para grafos em geral é difícil de se obter esse número. Então neste trabalho trabalhamos sobre uma classe de grafos denominada de grades parciais que, são sub grafos de grades, e descrevemos uma forma de obter o número de Grundy para essa classe. Neste trabalho também mostramos propriedades e estruturas que nos permitem construir uma solução algorítmica para esses tipos de grafos. Em especifico apresentamos duas soluções que podem ser utilizadas para encontrar o número de Grundy de grades parciais.
